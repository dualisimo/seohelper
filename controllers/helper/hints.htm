  • Google search console

    A good tool to help with websites optimization is Google Search Console. This tool allows you to track the indexing process and error messages, informs the owner about possible improvements etc. The tool also offers a possibility to load sitemaps or request index.

    Webpage of the tool is HERE

    For using this tool it is necessary to verify ownership with a verification file.

  • Google Companies

    A great opportunity to promote a webpage is also by creating a company profile on Google My Business. This option is particularly useful in cases of representing a company or a product with your optimized website. The option to create a profile on Google My Business is completely free with a possibility to include your position on a map.

    To verify your identity and page management competency there is a verification code sent to your address by post. With this code you are able to confirm your credibility. A combination of Google My Business and a Google+ account is yet another useful trick to get to the leading positions.

  • HTML Minification

    In the case of HTML website content minification interest the process is also simple. To apply you need to take these two steps:

    First it is necessary to turn minification on globally in the settings and then wrap the HTML code in {% spaceless %} and {% endspaceless %}.
    If interested in minifying the entire HTML code just wrap the whole layout file the same way.

  • Lowcost linkbuilding

    There is a variety of company lists in almost every country, free with basic options of creating a company profile with a short description and an address. The well known lists are marked as trustworthy for Google so it is wise and recommended to use this opportunity. Initial costs are very low but the impact can be very noticeable.
